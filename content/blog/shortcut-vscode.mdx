# Shortcut VSCode: Panduan untuk Meningkatkan Produktivitas

Visual Studio Code (VSCode) adalah salah satu editor kode paling populer di kalangan pengembang. Salah satu fitur yang membuat VSCode sangat efisien adalah shortcut keyboard-nya. Dalam artikel ini, kita akan menjelajahi shortcut VSCode yang paling berguna untuk meningkatkan produktivitas Anda.

<Image src="/images/blogs/vscode-shortcut.png" width="718" height="304" alt="Logo VSCode" />

## Mengapa Menggunakan Shortcut?

Menggunakan shortcut keyboard dapat secara signifikan meningkatkan kecepatan dan efisiensi Anda saat coding. Dengan meminimalkan penggunaan mouse, Anda dapat:

1. Menghemat waktu
2. Mengurangi kelelahan
3. Meningkatkan fokus pada kode

## Daftar Lengkap Shortcut VSCode

Berikut adalah daftar lengkap shortcut VSCode yang dibagi menjadi beberapa kategori:

| Shortcut                 | Fungsi                  |
| ------------------------ | ----------------------- |
