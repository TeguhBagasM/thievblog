# Belajar JavaScript Dasar

JavaScript adalah bahasa pemrograman yang powerful dan fleksibel, sering digunakan dalam pengembangan web. Bagi pemula, memahami konsep dasar JavaScript sangat penting untuk membangun fondasi yang kuat dalam pemrograman web. Mari kita jelajahi beberapa konsep fundamental JavaScript.

## Apa itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi, dinamis, dan interpreted yang terutama digunakan untuk membuat halaman web interaktif. Meskipun awalnya dikembangkan untuk browser web, JavaScript sekarang juga digunakan di banyak lingkungan non-browser, seperti Node.js untuk pengembangan server-side.

## Konsep Dasar JavaScript

### 1. Variabel dan Tipe Data

Variabel digunakan untuk menyimpan data. Di JavaScript, Anda dapat mendeklarasikan variabel menggunakan **let**, **const**, atau **var** (meskipun **var** sudah jarang digunakan dalam kode modern).

```javascript
let nama = "Budi";
const umur = 25;
let apakahMahasiswa = true;
```

### JavaScript memiliki beberapa tipe data dasar:

- String: untuk teks
- Number: untuk angka (bilangan bulat dan desimal)
- Boolean: true atau false
- Undefined: variabel yang belum diberi nilai
- Null: nilai kosong yang disengaja
- Object: untuk struktur data yang lebih kompleks

### 2. Operator

JavaScript mendukung berbagai jenis operator, cotoh:

```javascript
// Aritmetika: +, -, \*, /, %
// Perbandingan: ==, ===, !=, !==, >, <, >=, <=
// Logika: && (AND), || (OR), ! (NOT)

let x = 5;
let y = 3;
console.log(x + y); // Output: 8
console.log(x > y); // Output: true
```

### 3. Struktur Kontrol

## Mengapa Belajar JavaScript?

Populer: JavaScript adalah salah satu bahasa pemrograman paling populer di dunia.
Versatile: Dapat digunakan untuk front-end, back-end, mobile app, dan banyak lagi.
Komunitas Besar: Banyak sumber daya dan library tersedia.
Mudah Dimulai: Relatif mudah dipelajari untuk pemula.
