# Belajar Dasar Pemrograman Web: HTML dan CSS

## Apa itu Pemrograman Web?

Pemrograman web adalah proses pengembangan aplikasi dan situs web yang dapat diakses melalui internet. Ini melibatkan berbagai aspek, mulai dari pengkodean dan pengembangan hingga desain dan pemeliharaan.

Pemrograman web adalah dunia yang luas dan menarik, tetapi untuk memulai, kita perlu memahami dasar-dasar dari dua teknologi fundamental: HTML dan CSS. Dalam artikel ini, kita akan menjelajahi konsep-konsep fundamental dari dua teknologi ini dan bagaimana mereka bekerja sama untuk membuat website yang indah.

## Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. Ini adalah bahasa markup standar untuk dokumen yang dirancang untuk ditampilkan di peramban internet. HTML dapat dibantu oleh teknologi seperti Cascading Style Sheets (CSS) dan bahasa skrip lainnya seperti JavaScript, VBScript, dan PHP.
HTML digunakan untuk membuat struktur dasar halaman web, seperti mengatur teks, gambar, tautan, dan elemen-elemen lainnya. HTML tidak dianggap sebagai bahasa pemrograman karena tidak memiliki kemampuan untuk melakukan perhitungan atau logika.
HTML pertama kali diperkenalkan oleh Tim Berners-Lee pada tahun 1991. Sejak itu, HTML telah berkembang menjadi beberapa versi, dengan versi terbaru adalah HTML5.
HTML digunakan untuk membuat berbagai jenis halaman web, termasuk situs web pribadi, situs web bisnis, blog, dan aplikasi web.

## Apa itu CSS?

CSS adalah singkatan dari Cascading Style Sheets. Ini adalah bahasa pemrograman yang digunakan untuk menentukan bagaimana dokumen dan situs web akan disajikan. Dengan CSS, Anda dapat mengatur jenis font, warna tulisan, dan latar belakang halaman. CSS dibuat untuk dapat bekerja bersama dengan bahasa markup seperti HTML. Biasanya CSS digunakan untuk mengatur tampilan halaman. Ada tiga style di CSS, dan untuk menerapkan CSS di banyak halaman pada waktu bersamaan, gunakan External style.

### Contoh Kode HTML

Berikut adalah contoh kode HTML:

```html showLineNumbers title="index.html" caption="Ini adalah kode sederhana html"
<!DOCTYPE html>
<html>
  <head>
    <title>Belajar HTML</title>
  </head>
  <body>
    <h1>Selamat Datang di Website Saya!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>
```

### Penjelasan Kode HTML

Kode HTML di atas membuat halaman web sederhana dengan elemen-elemen sebagai berikut:

1.  **DOCTYPE html**: Menyatakan bahwa ini adalah dokumen HTML5.
2.  **html**: Elemen root dari dokumen HTML.
3.  **head**: Bagian yang berisi informasi meta, seperti judul halaman.
4.  **title**: Menampilkan judul halaman di tab browser ("Belajar HTML").
5.  **body**: Bagian utama yang menampilkan konten di halaman web.
6.  **h1**: Judul besar atau heading (dalam hal ini, "Selamat Datang di Website Saya!").
7.  **p**: Elemen paragraf yang berisi teks ("Ini adalah paragraf pertama saya.").

### Menambahkan CSS ke halaman HTML

Untuk menghubungkan halaman HTML dengan CSS, tambahkan baris berikut di dalam tag **head** atau dibawah **title**:

```html showLineNumbers
<link rel="stylesheet" href="styles.css" />
```

### Contoh Kode CSS

Berikut adalah contoh kode CSS sederhana:

```css
body {
  background-color: #f2f2f2;
  font-family: Arial, sans-serif;
}

h1 {
  color: #00698f;
  font-size: 36px;
}
```

### Penjelasan Kode CSS

1.  **"body {}"**: Bagian ini mengatur gaya untuk seluruh halaman (body).

    - **background-color: #f2f2f2;**: Mengubah warna latar belakang halaman menjadi abu-abu terang. Anda juga bisa menggunakan nama warna seperti **gray** atau kode warna seperti **#f2f2f2**.
    - **font-family: Arial, sans-serif;**: Mengatur font halaman menjadi **Arial**. Jika Arial tidak tersedia, browser akan menggunakan font alternatif dari keluarga **sans-serif**.

2.  **"h1 {}"**: Bagian ini mengatur gaya untuk elemen heading pertama (h1).

    - **color: #00698f;**: Mengubah warna teks untuk elemen h1 menjadi biru (tepatnya dengan kode warna heksadesimal **#00698f**). Anda juga bisa mengganti warna dengan nama warna seperti **blue**
    - **font-size: 36px;**: Mengatur ukuran teks untuk elemen h1 menjadi 36px (pixel), sehingga terlihat besar dan menonjol.

### Keterangan Tambahan

Warna dalam CSS dapat ditulis menggunakan dua cara utama:

- Nama warna: Seperti blue, red, green, dll
- Kode warna heksadesimal: Seperti #f2f2f2 untuk abu-abu terang atau #00698f untuk biru.

### Komentar dalam HTML dan CSS

**Komentar** adalah teks yang tidak akan tampil di halaman, digunakan untuk memberikan catatan atau penjelasan di dalam kode.

- Komentar dalam HTML ditulis dengan cara berikut:

```html
<!-- Ini adalah komentar HTML -->
```

- Komentar dalam CSS ditulis seperti ini:

```css
/* Ini adalah komentar CSS */
```

## Mengapa HTML dan CSS Penting?

> HTML dan CSS adalah pondasi dari setiap halaman web. Tanpa HTML, tidak ada struktur untuk ditampilkan, dan tanpa CSS, halaman web akan terlihat membosankan dan tidak menarik. Keduanya bekerja sama untuk menciptakan pengalaman pengguna yang menyenangkan.

### Hubungan Antara HTML dan CSS

HTML memberikan struktur, sedangkan CSS memberikan gaya. Anda dapat mengubah penampilan halaman web dengan mengedit kode CSS tanpa perlu mengubah struktur HTML. Misalnya, jika Anda ingin mengubah warna latar belakang atau jenis font, Anda hanya perlu memperbarui CSS, membuat proses pengembangan lebih efisien.
